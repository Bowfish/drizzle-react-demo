# Drizzle React Demo project


## Installation

Install all npm modules:

`$ npm install --save`

Install ganache:

`$ sudo npm install -g ganache-cli`


## Usage
Depoly the contracts:

`$ ganache-cli`

`$ cd src/eth`

`$ truffle migrate --reset`

Launch the dApp

`$ cd ../..`

`$ npm start`

Now you can access the app on http://localhost:3000/
